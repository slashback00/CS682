Python AirSim and ROS Code Documentation

Global Variables

velocity: Global variable to store the drone's velocity.

Functions

trajectory_callback(spline_marker_array_msg)

Callback function that follows the last published trajectory in the spline marker array.

Parameters:

  • spline_marker_array_msg: A 3d path matrix passed via a ROS message containing a marker array representing the trajectory path.

airsim_follow()

Main function to initialize ROS and set up subscribers.

Subscribers:

  • /trajectory/spline_marker_array: Subscribes to a ROS topic containing a marker array representing the trajectory path.

Main Execution

The code starts the airsim_follow() function as the main entry point for execution.
